# Amazon Connect API

Amazon Connect is a cloud-based contact center service that makes it easy to set up and manage a customer contact center and provide reliable customer engagement at any scale. Amazon Connect provides a seamless omnichannel experience through a single unified contact center for voice, chat, and task management. It is powered by machine learning to provide real-time analytics, sentiment analysis, and automated workflows.

## APIs

- **Amazon Connect API** - The Amazon Connect API provides programmatic access to manage contact center instances, users, routing profiles, contact flows, queues, and real-time and historical metrics for customer engagement operations.
